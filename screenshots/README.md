# Скриншоты для витрины

README ссылается на эти файлы. Сделай их на маке и положи сюда **с такими именами**:

- `about.png` — окно настроек, вкладка «О программе».
- `recognition.png` — вкладка «Распознавание».
- `terms.png` — вкладка «Термины» (покажи бренд-слова — это главная фишка).

Как снять: запусти приложение → значок «Qv» в строке меню → «Настройки…» → нужная вкладка → ⌘⇧4, пробел, клик по окну. Файл появится на рабочем столе, переименуй и перенеси в эту папку.

Пока файлов нет, картинки в README будут битыми — либо добавь их до `git push`, либо временно убери блок «Как выглядит» из README.
